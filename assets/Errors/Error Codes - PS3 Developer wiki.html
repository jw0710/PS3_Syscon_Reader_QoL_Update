What Syscon stores Was Syscon speichert

The PS3 Syscon maintains an error table with up to 32 entries. Each entry holds a 4-byte error code and a 4-byte timestamp. Der PS3 Syscon pflegt eine Fehlertabelle mit bis zu 32 Einträgen. Jeder Eintrag enthält einen 4-Byte-Fehlercode und einen 4-Byte-Zeitstempel.

Code format Code-Aufbau

Codes follow the structure A R ST C ERR, where each field adds context about timing, category, and fault type. Codes folgen der Struktur A R ST C ERR — jedes Feld liefert Kontext zu Zeitpunkt, Kategorie und Fehlertyp.

Why context matters Warum Kontext wichtig ist

A four-digit code alone is often not enough. The step number and surrounding log entries help determine whether a fault is the root cause or only a symptom. Ein vierstelliger Code allein reicht oft nicht aus. Die Step-Nummer und benachbarte Log-Einträge helfen zu bestimmen, ob ein Fehler Ursache oder Symptom ist.

Error Log Format Fehlerlog-Format
Timestamps are stored in J2000 time — seconds since 2000-01-01 12:00:00. If the battery was empty or removed when the error occurred, the timestamp may show as FFFFFFFF. Zeitstempel werden in J2000-Zeit gespeichert — Sekunden seit 2000-01-01 12:00:00. War die Batterie leer oder entfernt, kann der Zeitstempel als FFFFFFFF erscheinen.
J2000 timestamp behavior J2000-Zeitstempel-Verhalten
If the battery is replaced but the system time is not configured, timestamps can appear to start at an unusual date. This is normal behavior for an unset clock state. Wird die Batterie ersetzt, ohne die Systemuhr zu konfigurieren, können Zeitstempel an einem ungewöhnlichen Datum starten. Das ist normales Verhalten bei nicht gesetzter Uhr.
Why old logs can confuse diagnosis Warum alte Logs die Diagnose erschweren
The Syscon table can retain old errors for a long time. If diagnosing a current issue, it helps to clear the log first or compare repeated patterns across multiple readings. Die Syscon-Tabelle kann alte Fehler lange speichern. Bei der Diagnose eines aktuellen Problems empfiehlt es sich, das Log zuerst zu löschen oder wiederholte Muster über mehrere Auslesungen zu vergleichen.
Diagnostic Notes Diagnose-Hinweise

Multiple codes together Mehrere Codes zusammen

The surrounding codes matter. A 1002 next to 1601 or 3034 can mean something very different than a single 1002 by itself. Die umgebenden Codes sind entscheidend. Ein 1002 neben 1601 oder 3034 kann etwas ganz anderes bedeuten als ein einzelner 1002.

Timestamp sanity checks Zeitstempel-Plausibilitätsprüfung

If timestamps look strange, check battery state, clock configuration, and whether the log contains old unrelated entries. Wenn Zeitstempel seltsam aussehen: Batteriestatus, Uhrkonfiguration und alte, irrelevante Log-Einträge prüfen.

Step number context Step-Nummer-Kontext

The ST field tells you where in the boot sequence the failure happened — earlier steps suggest power or init problems, later steps suggest subsystem faults. Das ST-Feld zeigt, wo in der Boot-Sequenz der Fehler auftrat — frühere Steps deuten auf Strom- oder Init-Probleme hin, spätere auf Subsystem-Ausfälle.
